Avec le plugin [Starlight Sidebar Topics](https://github.com/hideoo/starlight-sidebar-topics) et la composante [Starlight Sidebar Topics Dropdown](https://github.com/trueberryless-org/starlight-sidebar-topics-dropdown), vous pouvez créer un site Web affichant une liste de thèmes sur le bureau et un menu déroulant sur mobile.

## Pré-requis

Vous aurez besoin d’un site web Starlight existant.

## Installez les paquets

Tout d'abord, installez le plugin `starlight-sidebar-topics` ainsi que la composante `starlight-sidebar-topics-dropdown` :

<Tabs syncKey="pkg">
  <TabItem label="npm">
    ```bash
    npm install starlight-sidebar-topics starlight-sidebar-topics-dropdown
    ```
  </TabItem>

  <TabItem label="pnpm">
    ```bash
    pnpm add starlight-sidebar-topics starlight-sidebar-topics-dropdown
    ```
  </TabItem>

  <TabItem label="Yarn">
    ```bash
    yarn add starlight-sidebar-topics starlight-sidebar-topics-dropdown
    ```
  </TabItem>
</Tabs>

Ensuite, suivez les guides d'installation pour [Starlight Sidebar Topics](https://starlight-sidebar-topics.netlify.app/docs/getting-started/#installation) et [Starlight Sidebar Topics Dropdown](https://starlight-sidebar-topics-dropdown.trueberryless.org/docs/getting-started/#installation).

## Modifiez la composante de la barre latérale

Dans le guide de configuration de la composante déroulante, vous avez créé une composante de barre latérale qui ne rend que le menu déroulant. Vous devez maintenant modifier cette composante pour qu'elle rende également la barre latérale standard lorsqu'un utilisateur est sur un bureau.

```astro
---
// src/components/Sidebar.astro
import Default from '@astrojs/starlight/components/Sidebar.astro';
import Topics from 'starlight-sidebar-topics/components/Sidebar.astro';
import TopicsDropdown from 'starlight-sidebar-topics-dropdown/TopicsDropdown.astro';
---

<div class="topics">
  <Topics/>
</div>
<div class="topics-dropdown">
  <TopicsDropdown/>
</div>
<Default><slot /></Default>

<style>
  /* Hide topics by default and show them on medium screens and larger */
  .topics {
    display: none;
  }

  @media (min-width: 800px) {
    .topics {
      display: block;
    }
  }

  /* Show topics dropdown on small screens and hide it on medium screens and larger */
  .topics-dropdown {
    display: block;
  }

  @media (min-width: 800px) {
    .topics-dropdown {
      display: none;
    }
  }
</style>
```

## Possibilités infinies

Vous pourriez également inverser les rôles (liste sur mobile, menu déroulant sur le bureau) en échangeant les propriétés CSS `display: block` et `display: none`.

De plus, vous pourriez créer votre propre composante d'affichage personnalisée, qui utilise les données de routage du plugin Starlight-Sidebar-Topics pour rendre les thèmes d'une manière personnalisée. C'est un peu plus avancé, mais vous trouverez plus d'informations à ce sujet dans la documentation ["Liste de sujets personnalisée"](https://starlight-sidebar-topics.netlify.app/docs/guides/custom-topic-list/).
